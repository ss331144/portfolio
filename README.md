## 📂 Project Overview

This repository contains a collection of Python scripts designed to:

- Automatically download YouTube video subtitles  
- Translate subtitles into Hebrew or other languages  
- Extract and process text from videos  
- Perform mathematical calculations such as prime number checking  
- Utilize advanced natural language processing (NLP) tools  

---

## 📦 Libraries Used

Here is a list of all Python libraries used in one or more scripts throughout the project:

| Library             | Description                                                  |
|---------------------|--------------------------------------------------------------|
| `pytube`            | Downloading YouTube videos and subtitles                     |
| `moviepy`           | Video editing and conversion                                 |
| `googletrans==4.0.0-rc1` | Automatic subtitle translation using Google Translate       |
| `os`                | Operating system interfaces (file management, commands)     |
| `re`                | Regular expressions for text processing                      |
| `math`              | Mathematical functions (e.g., primality tests)              |
| `sympy`             | Symbolic mathematics (derivatives, integrals, equations)    |
| `numpy`             | Advanced numerical operations and linear algebra            |
| `matplotlib.pyplot` | Data visualization and plotting                              |
| `subprocess`        | Running external system commands                             |
| `langdetect` *(optional)* | Language detection for translation workflows                |
| `speech_recognition` *(optional)* | Converting audio to text                                  |

> 💡 Tip: You can install all required libraries at once with:
> ```bash
> pip install pytube moviepy googletrans==4.0.0-rc1 sympy numpy matplotlib langdetect speechrecognition
> ```
